# Rungs — 엔트리로 배우는 AI

엔트리(Entry) 블록코딩 기반 AI 교육 프로젝트 10종을 소개하는 정적 웹페이지입니다.
추후 스크래치(Scratch), 파이썬(Python), AI 학습 콘텐츠로 확장 예정입니다.

## 배포

GitHub Pages로 배포되며, 이 저장소의 `main` 브랜치 루트(`/`)를 소스로 사용합니다.

## 구조

- `index.html` — 메인 페이지
- `브랜드로고/` — Rungs 심볼/워드마크 SVG
- `워크시트/` — 프로젝트별 MLforKids→Entry 변환 워크시트(.pdf)
- `워크시트/Entry files/` — 실제 엔트리 프로젝트 파일(.ent)

## 댓글(Cusdis) 설정

[cusdis.com](https://cusdis.com)에서 회원가입(구글 계정 등으로 가능) 후 사이트를 등록하면 App ID가 발급됩니다.
`index.html` 하단의 `#cusdis_thread` 요소에 있는 `data-app-id` 값을 그 App ID로 교체하면 댓글 기능이 활성화됩니다.
구글/페이스북/마이크로소프트 등 소셜 로그인으로 댓글을 남길 수 있어, GitHub 계정이 없어도 이용 가능합니다.
