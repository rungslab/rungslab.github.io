엔트리로 배우는 AI

블록코딩으로 만나는 10가지 인공지능 프로젝트 — 이미지 인식부터 데이터 예측까지!

🕵️

Secret Code

목소리로 스파이를 움직이는 음성인식 게임이에요. 말한 장소 이름을 인공지능이 알아듣고 스파이를 그곳으로 보내줘요.

🎤 음성인식 쉬움
🎯

Laser Eye

눈으로 조준하고 스페이스바로 쏘는 얼굴 인식 사격 게임! 고개를 움직여 목표물에 눈을 맞춰보세요.

😊 얼굴인식 보통
🐼

Shy Panda

누가 창문으로 쳐다보고 있는지 인공지능이 알아채는 부끄럼쟁이 판다예요. 웹캠 이미지로 사람이 있는지 판별해요.

📷 이미지인식 쉬움
🐼

Shy Panda — 얼굴 인식 버전

같은 부끄럼쟁이 판다지만 이번엔 이미지 분류 대신 얼굴 좌표 계산으로 창문 속 사람을 알아채요.

😊 얼굴인식 보통
🐾

가상 애완동물 (Flobzy)

배고프고 목마르고 슬퍼하는 나만의 반려동물! 웹캠으로 진짜 음식이나 물을 보여주면 AI가 알아보고 채워줘요.

📷 이미지인식 어려움
🎒

Journey to School

나이, 거리, 친구 수 같은 숫자 데이터로 학교 가는 방법을 예측하는 인공지능을 만들어요.

📊 데이터예측 쉬움
🏀

Catch the Ball

농구공이 날아가고 튕기는 모습을 물리 시뮬레이션으로 재현하고, 그 데이터를 모아 머신러닝 학습에 써봐요.

📊 데이터예측 어려움
🃏

Snap!

실제 카드를 웹캠에 보여주면 AI가 클럽·하트·다이아몬드·스페이드 무늬를 알아맞히는 카드 게임이에요.

📷 이미지인식 어려움
🐞

Shoot the Bug

돌아다니는 벌레를 네 가지 방식으로 맞혀보고, 쌓인 데이터로 머신러닝 조준 모델을 학습시켜요.

📊 데이터예측 어려움
😎

Face Finder

웹캠 속 내 얼굴 위로 만화 같은 눈과 코가 실시간으로 따라다니는 나만의 AR 얼굴 필터를 만들어요.

😊 얼굴인식 쉬움
